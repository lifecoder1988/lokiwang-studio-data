最近总有人问我，Claude 到底能不能干点「正经活」……

正好手上有个由头。我一直想找个足够刁钻的场子来压一压它：做一份车企的产品发布会 PPT。这活儿妙就妙在，它同时考三样东西——审美、信息组织、还有那股子发布会独有的「情绪」。差一点，就是村口大喇叭；对一点，才叫 keynote。

于是我给了它一个虚构的品牌：LUMEN，流明汽车，让它发一台旗舰电动轿车 L7。

然后我就翻了一遍它交出来的东西……说实话，第一屏亮出来那句 slogan，我就有点绷不住了。

01 开场之，「三年，一款车。」

发布会的第一屏，最见功力，也最容易翻车。

大部分人（和大部分 AI）会怎么写？「LUMEN L7 智能电动轿车，参数全面领先」——完了，通稿味扑面而来，观众已经开始摸手机了。

而它给的是这么两句：

三年，一款车。 从第一张草图到第一万台下线，LUMEN 只回答一个问题：电动车的下一个十年，应该长什么样。

……你别说，这就有那味儿了。一句短的砸下来立威，一句长的把「偏执」讲出来。收尾那句「为光而来」还顺手把 LUMEN（流明）这个名字给双关圆回来了。

就冲这个开场，我觉得这份 PPT 已经赢了一半。

02 硬参数之，它是真的懂「怎么吹」

发布会最难的，其实不是列参数，而是知道哪个数字该被放大。

这份 deck 里，它没有一股脑把 spec 全糊你脸上，而是一屏一个主角，还配了逐个跳动的数字动画：

  • 风阻 Cd 0.198，然后补一句「每降低 0.01 Cd ≈ 增加 8 km 续航」——你看，它知道光甩个 0.198 你没感觉，得翻译成续航你才会「哦——」；
  • 性能屏：综合最大马力 673 PS、零百 2.9 秒、最高车速 265 km/h，三个数字一屏，干净利落；
  • 三电屏：全域 900V 碳化硅高压平台、自研电控、三电一体化，术语堆得很「行活」。

最让我服的是补能那一屏。它没写「充电快」，它写的是：

10% → 80%，一杯咖啡的时间，补能约 500 公里。

以及一句「北京 → 上海，全程一次补能」。

这就是发布会话术的精髓：别报数字，报体感。一个 AI 能自己想到把「充电倍率」翻译成「一杯咖啡」，我是有点意外的。

03 座舱与智驾之，行业黑话它也会

往后翻，是这两年发布会的标配环节，它一个没落下，而且黑话说得很溜：

  • 座舱：3.2K 环幕三联屏（45 英寸一体式曲面）、车载大模型助手「全舱免唤醒、毫秒级连续对话」、手机·家居·车「一个账号无缝流转」；
  • 智驾：端到端一张神经网络、「从感知直达轨迹」、全国都能开、TOPS 算力平台、含激光雷达、海量真实场景训练数据。

「端到端」「一张神经网络」「全国都能开」……这几个词的排列组合，跟今年那几场真发布会几乎一个模子。它显然是把行业的语料喂明白了。

04 安全与价格之，最后一屏的「临门一脚」

安全屏走的是稳健路线：热成型钢笼车身、电池双面液冷、针刺/挤压/火烧全项通过、远端气囊 + 全舱侧气帘。中规中矩，但发布会该有的「安全感」给到了。

真正的临门一脚在最后：三色两版 + 价格。

  • L7 后驱长续航版：23.98 万起
  • L7 Max 四驱性能版：27.98 万
  • 「今晚 21:00 · 全渠道开启预订」

连「今晚 21:00 开启预订」这种制造紧迫感的收尾都安排上了……它是真把发布会的套路结构给学全了：立意 → 设计 → 性能 → 补能 → 座舱 → 智驾 → 安全 → 定价 → 逼单。一步不缺。

05 那，它到底行不行？

翻完我的整体感受是：信息组织和话术，接近能用的水平；纯审美，还得看你 prompt 下多少功夫。

它最强的地方，恰恰是最不像「AI」的地方——它懂克制。懂得一屏只讲一件事，懂得把参数翻译成体感，懂得开场用短句、收尾用双关。这些不是背参数能背出来的，是「懂发布会」。

当然也别急着吹。这毕竟是一台虚构的车，参数是它自己圆的，没有工程和供应链在后面兜底；真让它上台面对媒体的 Q&A，那又是另一回事了……（这里先按下不表）

但作为一份「从零到能放出来」的发布会 deck，我给的评价是：

发布会 PPT 这活儿，Claude 已经不是能不能做，而是你敢不敢直接拿去用了。

作品我放在下面了，建议自己划一遍——尤其看那个逐屏数字动画，比我这干讲有感觉多了。
