# LUMEN L7 发布会

一份用 **Claude** 生成的产品发布会 PPT / Keynote 网页，收录于 Work Show 作品集。

- 作品链接：<https://work-studio.baobaobao.online/works/lumen-l7>
- 形式：网页版滑动式发布会 deck（约 14 屏），带动效与逐屏数字动画。
- 主题：虚构的下一代智能电动旗舰轿车 **LUMEN L7**（流明汽车），2026 夏季发布会。

## 内容梗概（deck 各屏）

1. 开场：LUMEN · 流明汽车 / 下一代智能电动旗舰 / 「三年，一款车。」「为光而来」
2. 光刃美学：1.4 米贯穿式光刃日行灯、无框电吸门、半隐藏式激光雷达
3. 风阻：全球量产轿车第一梯队 Cd 0.198（每降 0.01 Cd ≈ +8 km 续航）
4. 性能：双电机全轮驱动、综合最大马力 673 PS、0–100 km/h 2.9 s、最高车速 265 km/h
5. 三电：全域 900V 碳化硅高压平台、自研碳化硅电控、三电一体化
6. 补能：10%→80% 一杯咖啡的时间、补能约 500 公里、900V 超充 vs 400V 平台、全国 2100+ 超充站；北京→上海 全程一次补能
7. 座舱：3.2K 环幕三联屏（45 英寸一体式曲面）、车载大模型助手全舱免唤醒毫秒级、手机·家居·车生态互联
8. 智驾：端到端一张神经网络、全国都能开、TOPS 算力平台、含激光雷达、海量真实场景训练数据
9. 安全：热成型钢笼式车身、电池双面液冷物理防护、针刺/挤压/火烧全项通过、含远端气囊与全舱侧气帘
10. 配置与价格：三色（极夜黑 / 流明银 / 曙光青）、两版（L7 后驱长续航版 23.98 万起 / L7 Max 四驱性能版 27.98 万）
11. 收尾：今晚 21:00 全渠道开启预订 · 「为光而来」

> 注：LUMEN L7 为演示用虚构产品，所有参数均为该 deck 的内容。
